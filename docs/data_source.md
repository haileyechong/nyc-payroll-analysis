# Data Source

Dataset: NYC Citywide Payroll Data  
Source: https://data.cityofnewyork.us/d/k397-673e

This dataset contains payroll records for New York City employees, including agency, job title, fiscal year, work location borough, base salary, regular gross pay, overtime hours, overtime pay, and other compensation.

The cleaned dataset was not pushed to GitHub because of file size. It is shared separately through Google Drive and should be placed at:

data/processed/nyc_payroll_cleaned.csv
